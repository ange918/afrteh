# afrteh - Next.js Application

## Overview
A Next.js 16 application with React 19, TypeScript, and Tailwind CSS 4.

## Project Structure
```
app/           - Next.js App Router pages and layouts
public/        - Static assets
```

## Tech Stack
- Next.js 16.1.6 with App Router
- React 19.2.3
- TypeScript 5
- Tailwind CSS 4
- ESLint for code quality

## Development
The development server runs on port 5000 with the command `npm run dev`.

## Deployment
Configured for autoscale deployment with:
- Build: `npm run build`
- Start: `npm run start`
