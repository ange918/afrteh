## TÂCHE 2 — SYSTÈME D'AUTHENTIFICATION (sans base de données)

Utiliser uniquement React Context + localStorage pour simuler l'auth.

### 2a. Créer src/context/AuthContext.tsx
```typescript
interface User {
  id: string;
  name: string;
  email: string;
  role: 'client' | 'styliste';
  avatar?: string;
  city?: string;
  specialty?: string; // pour les stylistes
  bio?: string;
  createdAt: string;
}
```
- AuthProvider avec useState<User | null>
- Fonctions :
  - login(email, password) → vérifie dans localStorage, retourne User ou erreur
  - register(name, email, password, role) → crée user dans localStorage, auto-login
  - logout() → clear state + localStorage
  - updateProfile(data) → met à jour le user
- Persister le user connecté dans localStorage ('dahomey_user')
- Exporter useAuth() hook
- Wrapper layout.tsx : <AuthProvider> autour de tout
